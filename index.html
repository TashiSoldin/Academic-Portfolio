Integrating Satellite Remote Sensing with Machine Learning for Ocean Health Assessment

University of Michigan - Capstone

Developed a machine learning pipeline to predict surface ocean dissolved oxygen concentrations from satellite remote sensing data for global ocean health monitoring. Integrated in-situ measurements from NOAA's World Ocean Database with NASA MODIS-Aqua satellite observations via Google Earth Engine API. Implemented comprehensive data processing including temporal filtering, deduplication, and spatial imputation to merge 125,000+ oceanographic measurements with coincident satellite data. Conducted feature engineering across seven categories (temperature transformations, spectral band ratios, normalized indices) and evaluated five ML architectures (Linear Regression, Random Forest, XGBoost, LightGBM, Neural Networks) with temporal cross-validation to prevent data leakage. Final XGBoost model achieved R² of 0.73 on geographically distinct test locations, demonstrating successful generalization of latitudinal oxygen gradients for potential early detection of ocean deoxygenation events.

Data Analytics Dashboard

University of Michigan - Milesone II

Developed machine learning models to predict dissolved oxygen concentrations in ocean waters using NOAA's World Ocean Database, achieving R² of 0.98 with an optimized Random Forest model. Processed 1.9M oceanographic observations through a systematic pipeline including cast-level salvageability analysis, intra-cast interpolation, and pressure-depth regression imputation to produce 602,900 complete records. Implemented and compared five model architectures (Linear Regression, Ridge, Random Forest, Gradient Boosting, Neural Networks) with comprehensive hyperparameter tuning and temporal cross-validation. Conducted in-depth failure analysis using SHAP feature attribution to identify three distinct failure mechanisms: depth-nutrient relationship mislearning, geographic undersampling bias, and data quality violations. Complemented predictive modeling with unsupervised clustering analysis (K-Means, Agglomerative, DBSCAN, HDBSCAN) to identify water mass similarities across spatial and chemical dimensions for ecosystem monitoring.

Climate Change & Mortality Analysis

University of Michigan - Milesone I

Conducted a comprehensive global analysis of the relationship between climate change and mortality using climate, population, and WHO mortality datasets spanning 130 countries. Designed and implemented a full data processing pipeline to harmonize climate (temperature, max temperature, precipitation) and mortality data across multiple sources, resolving country-level inconsistencies, restructuring wide-format datasets, engineering climate differentials, aggregating ICD-10 causes of death, and creating population-adjusted mortality metrics. Performed exploratory analysis and correlation studies to identify climate-sensitive demographic and cause-of-death categories, revealing stronger relationships with absolute temperature than with climate anomalies, particularly for older populations and circulatory, neoplasm, and mental health–related mortality. Delivered critical insights into data limitations, geographic biases, vulnerability patterns, and methodological considerations for future climate–health research.

Machine Learning Classification System

MIT ADSP - Capstone Project

Built a machine learning system to predict home-equity loan defaults for a financial institution, improving early risk detection and supporting more consistent, data-driven lending decisions. Conducted comprehensive exploratory data analysis, feature engineering, and model development using Logistic Regression, Decision Trees, Random Forests, and Gradient Boosting, including full hyperparameter optimization. Identified credit-history variables (delinquencies, derogatory marks, credit age, debt-to-income ratio) as the strongest predictors of default. Delivered a hybrid risk-tiering framework combining an optimized Random Forest for accuracy with an explainable Decision Tree for regulatory transparency, enabling targeted reviews, reduced default exposure, and improved underwriting efficiency.

Versatile Accelerated Digital Encryption Recovery (VADER) System

University of Cape Town - HPES

Designed and implemented an FPGA-based password recovery system for MD5-hashed passwords using parallel processing techniques as part of the High Performance Embedded Systems (HPES) course. Developed both dictionary and brute-force attack methods on a Nexys4 DDR board, achieving a speedup of 10.78x over sequential Python implementation. Created modular Verilog architecture with parallel supervisor modules managing multiple MD5 hashers to efficiently recover encrypted passwords.
