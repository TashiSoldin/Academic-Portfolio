Water Usage Prediction from Remote Sensing for Invasive Plant Management

University of Cape Town - Honours Thesis

Developed a machine learning model to predict water consumption of invasive alien plants threatening South Africa's water security using satellite remote sensing data. Combined Landsat 8 multispectral data with flux tower ground truth measurements to train five ML models (Neural Networks, RNN, LSTM, Random Forest, SVM) in estimating evapotranspiration. This Honours thesis work established the foundation for mapping invasive plant distribution and informing targeted management strategies across South Africa.

Versatile Accelerated Digital Encryption Recovery (VADER) System

University of Cape Town - HPES

Designed and implemented an FPGA-based password recovery system for MD5-hashed passwords using parallel processing techniques as part of the High Performance Embedded Systems (HPES) course. Developed both dictionary and brute-force attack methods on a Nexys4 DDR board, achieving a speedup of 10.78x over sequential Python implementation. Created modular Verilog architecture with parallel supervisor modules managing multiple MD5 hashers to efficiently recover encrypted passwords.

Verilog HDL FPGA Design Python Xilinx Vivado MD5 Hashing Parallel Processing Nexys4 DDR Hardware Acceleration
